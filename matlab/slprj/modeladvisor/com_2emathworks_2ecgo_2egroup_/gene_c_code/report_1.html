整数除法生成的代码可防范诸如除以零、INT_MIN/-1 和 LONG_MIN/-1 之类的算术异常。根据硬件的不同，算术异常可能会导致意外行为，因此针对这些异常提供保护非常关键。一些设计者以极其谨慎的方式构造其模型，这使得异常触发输入组合无法达到除法运算。对于这些模型，作为除法运算的一部分生成的保护代码是冗余的。可以使用 "配置参数" 对话框中的定点除法设置来删除这些冗余保护代码。仅当您严格地验证了模型不会在除法运算中导致异常时，才应采取此措施。
